# غذائيات الأخوة

موقع متجر **غذائيات الأخوة** — زيت زيتون وأعسال طبيعية ومنتجات بيتوتية، مع توصيل لباب البيت في سوريا وتركيا.

## التشغيل محلياً
```bash
python -m http.server 8000
```
ثم افتح `http://localhost:8000`.

## البنية
- `index.html` — الصفحة الرئيسية
- `styles.css` — التنسيق
- `app.js` — عرض المنتجات والحركة
- `src/data.js` — المنتجات والأسعار (مصدر واحد)
- `src/chat-widget.js` — مساعد المحادثة
- `assets/` — الصور
